# 微信小程序之本地商城

### 前言

自己练习的第二个小程序项目，无后台接口，数据为本地模拟，但功能较为完整

### 功能

+ 首页
  + 轮播图
+ 商品详情页
  + 加入购物车动画
  + 数量加减
  + tab栏切换详情
  + 跳转购物车页面
+ 购物车
  + 全选功能
  + 计算总价功能
  + 删除商品
  + 数量加减
  + 跳转支付页面
+ 添加地址
  + 保持数据存储返回原页面
+ 分类
  + 侧边栏显示二级菜单
